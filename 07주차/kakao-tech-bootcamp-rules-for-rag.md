## 1. 출결 규칙 (Attendance)

### 1.1 출석 용어

| 용어 | 정의 |
|------|------|
| 정상 출석 | 일 8시간 교육 참여 |
| 결석 | 지각·외출·조퇴 등으로 일 4시간 미만 교육 참여 (점심시간 제외) |
| 지각 | 오전 09:10 이후 출석체크 시 |
| 외출 | 교육시간 중간에 일정 시간 동안 참여 불가한 경우 |
| 조퇴 | 교육시간 중간부터 끝까지 참여 불가한 경우 |

### 1.2 수료 조건

- **전체 교육 일수의 80% 이상 출석**해야 수료 가능
- 지각·조퇴·외출 **3회 누적 시 1일 결석** 처리
- 하루에 지각·외출·조퇴가 **4시간 이상**(점심 제외)이면 해당일 결석 처리
- 1시간 = 1교시(교육시간 + 쉬는시간) 포함

### 1.3 출석 방법 (HRD-Net QR 코드)

- **사전 준비:** HRD-Net 앱 다운로드 → 로그인 → 실명 인증
- **입실 (08:50~09:10):** HRD-Net 앱 → QR 버튼 → 입실 → ZOOM 내 QR 스캔
- **퇴실 (17:50~18:10):** HRD-Net 앱 → QR 버튼 → 퇴실 → 디스코드 내 QR 스캔
- 퇴장하기 미진행 시 해당일 결석 처리
- 일반 QR 스캐너(네이버/카카오) 아닌 **HRD-Net 앱 자체 스캐너** 사용 필수
- 입/퇴장시간 시스템 자동 등록, **개인 불찰 수정 불가**
- HRD-Net QR 출결은 운영팀도 정정 불가 (고용센터 실시간 기록)

### 1.4 외출/조퇴 절차

- **외출:** 출석 → 외출 출발 → 복귀 → 퇴실 (총 4회)
- **조퇴:** 출석 → 퇴실 (총 2회)
- 미입력 적발 시 **부정출결 → 해당일 결석 처리**

### 1.5 출석 인정 (공가)

- **근거:** 국민내일배움카드 제34조 제2항 제2호 [별표3] 해당 사유
- **신청 시한:** 평일 기준 해당일 **3일 전까지** 통합신청센터 제출 (경조사만 당일 가능)
- **변경/취소:** 해당일 **1일 전까지** 통합신청센터
- **증빙:** 증빙자료 함께 제출, 어려울 시 디스코드 출석문의 채널 활용
- **휴가:** 6월부터 사용 가능 (또는 수강철회 기간 이후)

### 1.6 공가 제한

- 공가(질병)는 **전체 소정훈련일수의 10%**까지만 가능 (단위기간 기준 아님)
- 잦은 질병 신청은 헬퍼라이언 면담 필요
- 미용 목적 치과/피부과 진료는 공가 사유 **불가**

### 1.7 출석 확인 방법

- HRD-Net 앱에서 본인 출결 현황 직접 확인 가능

## 11. 페이지 구조 (RAG 청킹 참고)

```
메인: [카카오테크 부트캠프] 4기
├── 크루 온보딩 가이드 (하위 페이지)
├── OT 온보딩 가이드 (하위 페이지)
│   ├── 탄생 안내 (하위 페이지)
│   ├── 출석 기준 진행 방법 (하위 페이지)
│   └── 훈련 장려금 (하위 페이지)
├── 풀스택 (트랙 페이지)
├── AI 실무 개발 (트랙 페이지)
├── 클라우드 네이티브 (트랙 페이지)
├── 수료 D-152 (하위 페이지)
├── 📢 공지사항 (DB, 6개 항목)
├── 🚀 내비게이션 (8개 바로가기)
├── 💾 출결 관련 서류 (탭, 10개 항목)
├── 🫵 교육일정 & 규정 (탭, 6개 항목)
├── ⛑️ 지원 (탭, 5개 항목)
└── 이모저모 (탭, 빈 DB)
```

> **퍼블릭 접근 제한 참고:**
> - 출결 관련 서류 탭의 10개 항목, 교육일정 & 규정 탭의 세부 항목(경고 및 제적 기준 등), 지원 탭의 5개 항목은 **Notion DB 내부 항목**으로, 퍼블릭 뷰에서는 **제목만 확인 가능**하고 상세 내용은 확인 불가
> - 이모저모 탭은 빈 DB
> - 공지사항 DB는 최근 6개 항목 확인 가능
